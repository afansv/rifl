Почему Rifl?

Автоудаление

Заметки на Rifl удаляются сразу же после прочтения. Если страница заметки не была посещена в течении суток, она будет удалена.

Анонимность

Rifl не ведёт логи конфиденциальных данных, не собирает IP-адресса, User Agent и прочие данные в целях сбора статистики или передачи третьим лицам.

Открытый исходный код

Исходный код Rifl открыт для всех желающих, это значит, что каждый может проверить сервис на наличие бекдоров или установить копию на любой другой сервер.

Публичный API

Наш API позволяет всем желающим создавать клиенты Rifl для самых разных платформ. Документацию к API можно прочитать вот здесь.
